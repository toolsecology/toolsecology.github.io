How to Embed a Tool

This can be done in 3 Simple steps:

1 - Configure the tool parameters. You can for instance choose the frame color that best matches your site.

2 - Click the "Copy to clipboard" button located at the bottom of the tool. This will copy the embedding code in your clipboard so you can paste it elsewhere.

3 - Choose where you would like to place the tool. This could be in an HTML page or a Wordpress site:

HTML page

- Just paste the code (Command-V for Mac users, Ctrl+V for Windows users) where you want the tool to appear.

And you're done! When you load the page, you should now see the tool appear. 

Wordpress site

- When editing a post or page select "Add block". 

Choose a "Custom HTML" block (you might have to click "Browse all" to see this option). Paste the code (Command-V for Mac users, Ctrl+V for Windows users) in the "Write HTML" window.  And you're done! When you click preview, you should now see the tool appear.
